Local-first AI navigation assistant. Fine-tuned Microsoft Phi-3 Mini with QLoRA, a custom RAG pipeline called Truth Anchor that injects verified JSON data to prevent hallucinations, and an intent classification shield to filter queries before the ranking engine. Runs fully on-device with 4-bit quantization.
